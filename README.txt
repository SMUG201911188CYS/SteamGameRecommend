프로그램 이름: 스팀 게임 추천기

제작자 - 전선현, 최용성, 황윤신, 정태송

개발 기간: 2023.11.13 ~ 2023.12.17

[ 프로그램 설명 ]
  - 스팀 게임의 태그를 이용하여 사용자가 선호하는 게임을 찾고, 게임에 대한 정보를 보여준다.
  - 사용자는 추천받은 게임의 이름, 헤더 이미지, 설명, 출시일, 가격, 지원 언어, 게임 스코어에 대한 정보를 제공받는다.
  - 사용자는 태그, 추천받을 게임 수, 인기도를 선택할 수 있다.
  - 태그는 스팀에서 제공하는 태그를 선택할 수 있다.
  - 추천받을 게임 수는 steam_games.csv 데이터셋이 가지고 있는 게임 수만큼만 설정 가능하다.
  - 인기도는 steam_games.csv 데이터셋에서 제공하는 게임의 리뷰 수를 기반으로 한다.

[ 프로그램 실행 방법 ]
  1. 명령 프롬프트 실행
  2. 명령 프롬프트에서 'pip install flask'를 입력하여 flask 설치
  3. Steam_Game_Recommendation_Machine 폴더 압축 해제
  4. 명령 프롬프트에서 cd 명령어를 이용하여 폴더에 접근
  5. 명령 프롬프트에서 'python app.py'를 입력하여 app.py실헹
  6. 인터넷 브라우저의 URL 검색창에 'http://127.0.0.1:5000/' 입력하여 접속
